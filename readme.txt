Здравствуй пользователь! Данный проект подразумевает чат бота для Telegram. Работает бот на основе небольшой базы данных о Солнечной системе. Выводит данные из БД в доступной и удобной форме.
____________________________________________________________________________________
                                 ИНСТРУКЦИЯ О ЗАПУСКЕ

1. Перед запуском бота откройте файл "config.py" и отредактируйте в нём параметр "TOKEN", вставив свой токен чат бота Telegram(токен создается с помощью @BotFather);
2. Откройте файл "ChatBot.py" и запустите его;
3. После этого перейдите уже в созданного бота(с помощью @BotFather в Telegram) и пропишите "/start".
____________________________________________________________________________________
                                 ИНФОРМАЦИЯ О ФАЙЛАХ
1. База данных находится в файле "solar_system.sqlite";
2. Техническое задание находится в файле "ТЗ.txt";
3. Чат бот находится в файле "ChatBot.py";
4. Чтобы поменять токен откройте файл "config.py" и отредактируйте параметр "TOKEN";
5. Инструкция(вы ее сейчас читаете) находится в "readme.txt";
6. Чтобы узнать все используемые модули в проекте откройте файл "requrements.txt";
7. Презентация о данном проекте находится в файле "About_SolarSystem_ChatBot.pptx".
